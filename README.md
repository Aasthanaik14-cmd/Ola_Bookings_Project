This project focuses on analyzing Ola ride bookings using SQL for data cleaning, transformation, and analytical queries. Key insights include ride trends, cancellation reasons, customer behavior, payment patterns, and vehicle-type performance.
Power BI was used to build an interactive dashboard showcasing KPIs, charts, slicers, and breakdowns.
The project highlights real-world data analysis skills, including SQL querying, DAX measures, and creating visually driven business insights.
