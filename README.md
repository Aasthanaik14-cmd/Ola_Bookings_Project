This project focuses on analyzing Ola ride booking data by combining machine learning, SQL, and Power BI. A Linear Regression model was built to predict ride prices based on factors like distance, duration, and vehicle type. SQL was used for data cleaning and exploratory analysis to identify trends such as ride volumes, peak hours, and vehicle performance. The results were visualized in an interactive Power BI dashboard, including actual vs predicted comparisons and error analysis, providing meaningful business insights and evaluating model accuracy.
